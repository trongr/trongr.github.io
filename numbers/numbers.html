Number Theory

Bezout's Identity

Lemma. Let d be the greatest common divisor of nonzero integers a and b. Then there exist integers x and y s.t. $$ax + by = d.$$

Congruences

Definition. If a nonzero integer $m$ divides $a-b,$ then we say that $a$ is congruent to $b$ modulo $m$ and write $$a \equiv b \mod m.$$

Theorem.

  • Congruence respects addition and multiplication: if $a \eq b \mod m$ and $c \eq d \mod m,$ then \begin{align*} a + c &\eq b + d && \mod m \\ ac &\eq bd && \mod m. \end{align*}
  • If $a \eq b \mod m$ and $d \mid m,$ then $a \eq b \mod d.$
  • If $a \eq b \mod m$ and $c \neq 0,$ then $ac \eq bc \mod mc.$

Note. We're allowing the divisor $d$ and $mc$ to be negative.

Corollary. Let $f$ be a polynomial with integer coefficients. If $$a \eq b \mod m$$ then $$f(a) \eq f(b) \mod m.$$

Theorem.

  • $ax \eq ay \mod m$ iff $x \eq y \mod \frac{m}{(a, m)}.$
  • Let $(a, m) = 1.$ Then $ax \eq ay \mod m$ iff $x \eq y \mod m.$
  • $x \eq y \mod m_i$ for $i=1,\ldots,r$ iff $x \eq y \mod [m_1,\ldots,m_r].$

Complete residue system

Definition. If $x \eq y \mod m,$ then $y$ is called a residue of $x$ modulo $m.$ A set $x_1,\dots,x_m$ is called a complete residue system mod $m$ if for every integer $y$ there is an $x_j$ s.t. $y \eq x_j \mod m.$

A set of $m$ integers forms a complete residue system mod $m$ iff all integers in the set are distinct mod $m.$

Theorem. If $b \equiv c \mod m,$ then $(b, m) = (c, m).$

Fermat's Theorem

Theorem. Let $p$ be a prime. If $p \nmid a,$ then $$a^{p - 1} \equiv 1 \mod p.$$ For every integer $a,$ $$a^p \equiv a \mod p.$$

Euler's Totient Function

Definition. The number $\f(m)$ is the number of positive integers less than or equal to m which are relatively prime to m.

Euler's generalization of Fermat's theorem

Theorem. If $(a, m) = 1,$ then $$a^{\f(m)} \equiv 1 \mod m.$$

Lemma. Let $p$ be prime. Then $$x^2 \equiv 1 \mod p \quad\text{ iff }\quad x \equiv \pm 1 \mod p.$$

Wilson's Theorem

Theorem. If $p$ is a prime, then $$(p - 1)! \equiv -1 \mod p.$$ Conversely, if n is not prime, then $$(n - 1)! \not\equiv -1 \mod n.$$

Proof of converse. Suppose $n$ is not prime, with prime factor $p.$ Then $p$ appears in the factorial $(n - 1)!,$ so $(n - 1)! \equiv 0 \not\equiv -1 \mod p.$ Since $(n - 1)! + 1$ is not divisible by $p,$ it cannot be divisible by $n.$

Theorem. Let p be a prime. Then $x^2 \equiv -1 \mod p$ has a solution iff $p=2$ or $p\equiv 1 \mod 4.$

Lemma. If p is a prime and $p\equiv 1\mod 4,$ then there exist positive integers a and b s.t. $a^2 + b^2 = p.$

Lemma. Let q be a prime factor of $a^2 + b^2.$ If $q \equiv 3 \mod 4,$ then $q|a$ and $q|b.$

Theorem (Fermat). Write the canonical factorization of n in the form $$n = 2^\a \prod_{p\equiv 1(4)} p^\b \prod_{p\equiv 3(4)} q^\g.$$ Then n can be expressed as a sum of two squares iff the exponents $\g$ are even.

Exercise. Let $f$ be a polynomial with integral coefficients and $f(a) \equiv k \mod m.$ Then $f(a + tm) \equiv k \mod m$ for every integer t.

Exercise. $19$ is not a divisor of $4n^2 + 4$ for any integer n.

Use the fact that for a prime $p,$ $x^2 \equiv -1 \mod p$ has a solution iff $p=2$ or $p\equiv 1 \mod 4.$

Exercise. $7|(3^{2n+1} + 2^{n+2}).$

Rewrite $3^{2n+1} + 2^{n+2} = 9^{n}3 + 2^{n}4 \equiv 2^{n}3 + 2^{n}4 \equiv 2^{n}(3 + 4) \equiv 0 \mod 7.$
