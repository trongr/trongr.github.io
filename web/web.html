Useful Curl Commands

GET 
curl -X GET "http://example.com/asdf?id=abc"
POST 
curl -X POST --data "" "http://example.com/asdf"
curl -X POST --data "id=abc&token=123" "http://example.com/asdf"
curl -X POST --data-urlencode "id=abc" --data-urlencode "callback=http://asdf.com/callback" "http://example.com/asdf"
DELETE 
curl -X DELETE --data "id=abc&token=123" "http://example.com/asdf"
